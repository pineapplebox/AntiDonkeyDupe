# **AntiDonkeyDupe**

#### A simple protection against the Donkey Dupe on Minecraft servers

## **Dependencies:**
#### ProtocolLib 4.5.0 or higher


## **Compiling**

**Make sure that Maven and Git are already installed**

```
git clone https://github.com/pineapplebox/AntiDonkeyDupe.git
```

```
cd AntiDonkeyDupe
```

```
mvn clean package
```
